Uxaria is a user interface toolkit for the web based on Twitter Bootstrap. It shares the naming conventions for classes and variables with Bootstrap, but differs in design and implementation.

Uxaria provides a modular set of sane default styles for most common elements in complex web designs. Responsive design is at the core of this project, with the flexibility for adaptions that make sense in mind.

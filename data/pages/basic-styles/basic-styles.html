Font

The default font size is set to 0.9375em – corresponding to 15px in most browsers. This font size is used for mobile phones up to 568px, for tablets in portrait orientation with 768px device width and for tablets and small desktops with 1024px device width.

For larger screens the font size is adjusted to fill the device or browser window width up to a maximum of 1.25em (corresponding to 20px) on a 1366px wide screen.

The default line height is 1.4 - a proven value for good readability for lines with approximately 80 characters per line. The grid system is optimized for this line length at 6 grid columns.

Uxaria includes the marvelous CartoGothic Std font family by FontSite. It's very legible and clean free font similar to Frutiger.
